Employer and institution lookup

Verify an Odyssey certificate without entering the student portal.

Enter the student's legal name and 12-digit certificate code to confirm the certificate, course completion, instructor, issuing institution, dates, and the learner's verified Odyssey education history.

Code patternThe first digit identifies the education category. The second digit identifies the discipline family.
Name matchThe lookup requires the student's name and certificate code together to reduce casual scraping.
